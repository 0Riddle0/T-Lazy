# T-Lazy
A script for all the Termux script-kiddies out there.

## Installation:

`git clone https://github.com/Ethical-Hacking-Zone/T-Lazy.git`

`cd T-Lazy && chmod +x install.sh && ./install.sh`

## Running Instructions:

`python3 T-Lazy.py`
